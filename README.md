Online recruitment platforms are increasingly targeted by fraudsters who post fake job advertisements to scam job seekers.  
This project proposes a deep learning based system to detect fraudulent job postings using Natural Language Processing (NLP) techniques and transformer-based models such as BERT and RoBERTa.

The system classifies job postings as **Real Job** or **Fraudulent Job** based on textual content and related features.
